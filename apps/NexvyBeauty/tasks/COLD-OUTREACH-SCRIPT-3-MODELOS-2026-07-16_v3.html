1A tensão Desconfiada × Queimada — resolvida

As duas querem o oposto: a Desconfiada bloqueia se você mostra saber algo das clientes dela; a Queimada só responde se você prova, concreto. Conflitam só enquanto "prova" = "posse dos dados dela".

Destravamento — a prova não é posse, é AUTO-REVELAÇÃO: o dado é SEU, quem revela é o seu próprio WhatsApp, eu não tenho acesso a nada — eu só te ajudo a enxergar.
PersonaComo o frame "É Seu" resolve
Desconfiadanão reivindico posse; está no celular dela → some o gatilho de bloqueio
Queimadaé o dado DELA (checável) + mecanismo concreto + "você vê", não "eu prometo"
Limite honestoo ideal pleno da Queimada (nome real na 1ª linha) é impossível no frio sem invadir → ela fecha no raio-x, não na abertura

Conclusão: UMA família de abertura ("É Seu") serve as 4; o cínico é conversão-de-demo. As 3 variantes mudam só o objeto (lista/quanto/quantas), todas posse-zero.
